Create Jam

September 2019

On this project I was UI designer and product owner on an app for the game jam organisation Create Jam. The purpose of the app was to allow participants to browse, favourite and keep track of games created during the jam, as well as submitting their vote for the best game.

I was in the Create Jam committee at the time, and became product owner on the project. This meant that I had to prioritise the features that we would like in the app. When we had agreed on a design, I created this prototype, as a visual guide of what the app should look like.
