    Naturbasen

    UI Redesign

    Date: March 2020

    Categories: Design, Prototyping

    "Naturbasen" is an app which allows the user to report observations about the danish flora and fauna, as well as providing an overview of animals and special habitats in the vicinity.

    This case is a graphical redesign of the existing app, which came up as a suggestion following a coffee meeting I had with a company. The result is a prototype created using Figma, with a focus on the graphical aspects of the app, as well as some functional changes, which contribute to a better user experience.

    Create Jam

    UI Design

    Date: November 2019

    Categories: Scrum, Graphical Design, Prototyping

    This case considers an app for Create Jam, which is an organisation which hosts events, where people of different creative backgrounds get together to create games over a weekend. The app's primary use case is to browse the games create during the event, and to enable voting on the games. The winner takes home the main prize.

    As an active member of the organisation together with my background, it was natural fit for me to be the product owner on the case. AS product owner I had the responisbility to prioritize the features which create the most value for the users. I was also responsible for the graphical design and create a design guideline for the developers to follow. My technical background made commnunication between me and the developers run smoothly, regarding the technical considerations of the product.
